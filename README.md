# 介護事業者向け 重要事項説明動画DX ランディングページ

介護施設・介護事業所向けの「重要事項説明書・利用契約書 説明動画制作サービス」のランディングページです。
ご家族に伝わりやすい契約説明、職員の説明負担軽減、説明品質の標準化、契約時の認識違い防止を訴求し、申込フォームへ誘導します。
